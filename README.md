# Awtoyoly Vault

IT Infrastructure Password & Credential Manager - built for network/system administrators.

## Features (Planned)
- Multi-user with RBAC (Admin, Manager, Viewer)
- 2FA (Google Authenticator / TOTP)
- AES-256 encrypted credential storage
- Branch/Location based organization
- Device type icons (Server, Firewall, Switch, NVR, PBX, ESXi)
- One-click SSH/RDP/HTTPS launch buttons
- Password strength meter
- Audit log (who viewed/changed what)
- Password history
- Export (Excel/PDF)
- Dark modern UI
- SQLite + Flask + JWT

## Tech Stack
- Python / Flask
- SQLite (encrypted fields with AES-256)
- JWT authentication + TOTP 2FA
- Single-page dark UI
